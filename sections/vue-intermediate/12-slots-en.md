# Slots

## Slots

enable creation of components like these:

```xml
<button-primary>submit</button-primary>
```

```xml
<dialog-modal>
  <h1>About</h1>
  <p>Todo App by Marko</p>
  <button>close</button>
</dialog-modal>
```

## Slots

`primary-button` component template:

```xml
<button class="btn-primary">
  <slot></slot>
</button>
```
