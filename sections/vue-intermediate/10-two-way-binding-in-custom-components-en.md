# Two-way binding in custom components

## Two-way binding in custom components

two-way binding for inputs:

```xml
<input v-model="newTitle" />
```

two-way binding in custom components:

```xml
<star-rating v-model="productRating" />
```

```xml
<todo-item
  v-model:title="todo.title"
  v-model:completed="todo.completed"
/>
```
